# 说明

基于 nodejs 18.18.2 进行开发

# 项目初始化过程

npm i -g pnpm
pnpm i -g vite@5
pnpm create vite@5
pnpm i vue-router
pnpm i element-plus

# 运行项目

pnpm run dev

# 环境说明

## 开发环境（临时）

- pnpm run temp

## 开发环境（云桌面）（暂未部署）

- pnpm run dev
- pnpm run build:dev

## 生产环境（终端）

- pnpm run prod
- pnpm run build:prod
